## How to install the application:

### Requirements:
1) Flutter installed in your laptop with Dart too.
2) Android Studio flutter plugins and android emulator.

### Installing application:
Firstly, extract the code from “ipfv.zip”.
1) Through Command Prompt
- Open cmd from start menu and go to the location of file IPFV.
- apply command "flutter run ipfv".
2) Through VS-Code
- Open file of IPFV in VS-Code.
- select "debug->start without debug" or simply press "ctrl+F5" to run the project.
